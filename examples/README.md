# Examples using FiniteFlow with Python

This a collection of examples and tests using the Python interface of FiniteFlow:

- [`ibp_reduction`](ibp_reduction/README.md) contains an example performing IBP reduction and reconstructing differential equations for the master integrals of a loop integral family.  It requires an external program to generate the identities and the derivatives of loop integrals (we use Mathematica and CALICO in the example).

- [`cas_interfaces`](cas_interfaces/README.md) contains examples using FiniteFlow with SymPy or SymEngine
