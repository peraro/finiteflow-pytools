FiniteFlow PyTools
==================

A collection of [tools](packages/README.md) and [examples](examples/README.md) using the Python interface of [FiniteFlow](https://github.com/peraro/finiteflow).

These currently require the `exp` branch of FiniteFlow and its [Python interface](https://github.com/peraro/finiteflow/blob/exp/pythonapi/README.md) to be installed.
